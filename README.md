# AI爱好者个人主页

静态个人主页，定位为“AI 爱好者的副业/变现实验记录”。

## 编辑内容

- 平台链接和统计数字：修改 `script.js` 里的 `siteConfig`
- 页面正文：修改 `index.html`
- 视觉样式：修改 `styles.css`
- 主视觉：替换 `assets/ai-lab-dashboard.png`

## 本地预览

直接打开 `index.html` 即可。需要模拟线上路径时，可在本目录启动任意静态服务器。

## GitHub Pages

仓库发布到 GitHub 后，使用 GitHub Pages 从 `main` 分支根目录发布。
